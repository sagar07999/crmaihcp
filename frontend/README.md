# AI First CRM HCP Module

## Tech Stack

- React
- Redux
- FastAPI
- LangGraph
- Groq (Gemma2-9B-IT)
- PostgreSQL/MySQL

## Features

- Log Interaction Form
- AI Chat
- AI Summarization
- CRUD
- LangGraph Agent
- Five Sales Tools

## Run Backend

```bash
cd backend

pip install -r requirements.txt

python -m uvicorn app.main:app --reload
```

## Run Frontend

```bash
cd frontend

npm install

npm run dev
```
