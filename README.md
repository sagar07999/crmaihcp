# AI First CRM HCP Module

## Tech Stack
- React
- Redux
- FastAPI
- LangGraph
- Groq LLM
- PostgreSQL
- SQLAlchemy

## Features
- Log Interaction
- AI Chat
- Search Interactions
- Edit Interaction
- Delete Interaction
- AI Summary
- Follow-up Suggestions
- Next Visit Recommendation

## Backend

cd backend
python -m venv venv

pip install -r requirements.txt

uvicorn app.main:app --reload

## Frontend

cd frontend

npm install

npm run dev
